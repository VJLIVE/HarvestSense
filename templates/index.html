About the Project

This project helps farmers, agricultural experts, and policymakers predict crop yield based on key factors such as:

  • Year: The year of data collection.
  • Average Rainfall: The annual precipitation in millimeters.
  • Pesticide Usage: Amount of pesticides used (in tonnes).
  • Average Temperature: Yearly temperature in degrees Celsius.
  • Area: The geographical region where crops are grown.
  • Item: The type of crop being cultivated.

How It Works

The system uses a Decision Tree Regressor model trained on historical agricultural data. The process involves:

  1. Data preprocessing: Handling missing values and encoding categorical data.
  2. Training a Machine Learning model to predict yield.
  3. Deploying the trained model using Flask.
  4. Providing a web interface for users to input data and get predictions.

Dataset Overview

The dataset includes agricultural data from various countries and years, cleaned and preprocessed for model accuracy.

  • Total Records: 10,000+ data points
  • Source: Open-source agricultural databases
  • Features: 6 input features affecting crop yield
